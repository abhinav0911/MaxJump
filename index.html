MaxJump: A Strategy Game

Game Rules...

  • Opponents:
    1. Human - 'O'
    2. AI - 'X'
  • Moves:
    1. Click source cell, then target cell.
    2. Target cell must be empty.
    3. Jump straight over opponent to capture.
    4. While capturing, jump size=2; else jump size=1.
    5. For upper and lower triangular analogies, possible moves are along green lines:
  • Result: If all troops get captured, opponent will win. Tie Game (3 min).
